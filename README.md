# WebLynk Landing Page

A modern, responsive landing page built using **HTML5**, **CSS3**, and **JavaScript**. The project is designed to demonstrate clean UI design, semantic HTML, responsive layouts, and interactive user experiences.

## 🚀 Features

- Responsive navigation bar with mobile menu
- Hero section with call-to-action
- About section
- Course showcase using CSS Grid
- Features section
- Contact form
- Responsive footer with social links
- Smooth hover animations
- Semantic HTML5 structure
- Mobile-first responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome

## 📱 Responsive Design

Optimized for:

- Desktop
- Tablet
- Mobile
